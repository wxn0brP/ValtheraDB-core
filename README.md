# @wxn0brp/db-core

Core library for [ValtheraDB](https://github.com/wxn0brp/ValtheraDB).

## ⚠️ Disclaimer

It is **not recommended** to install this module separately unless you plan to create your own ValtheraDB adapter.

## Installation

```bash
npm install @wxn0brp/db-core
```
